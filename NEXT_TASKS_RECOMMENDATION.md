**DB 스키마 추가**:
```sql
CREATE TABLE notifications (
    id UUID PRIMARY KEY,
    user_id VARCHAR,
    type VARCHAR,  -- 'workflow_success', 'workflow_failed', 'system_warning'
    title VARCHAR,
    message TEXT,
    read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP
);
```
